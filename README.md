# Test1

Statische Website für **babixGO** (Monopoly GO Services). Inhalte werden direkt in HTML/CSS gepflegt.

## Projektstruktur (Kurzüberblick)
- `index.html`: Startseite mit Angebotsblöcken und Navigation.
- `anleitungen/`: Anleitungsseiten (z. B. Freundschaftsbalken-Anleitung).
- `assets/`: Styles, Icons, Bilder.
- `downloads/`: Download-Seite(n).
- `public/`: Weitere statische Inhalte.
- `robots.txt`, `sitemap.xml`: SEO-Dateien.

## Lokales Öffnen
- Direkter Aufruf der `index.html` im Browser.
- Optional: lokaler Static-Server, z. B. `python -m http.server` im Repo-Root.
